## Features
- Comprehensive documentation.
- User-friendly interface.
- Efficient management of physical therapy workflows.

## Installation
To install Fisio Flow Care, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Ewrogabriel/fisio-flow-care.git
   ```
2. Navigate into the project directory:
   ```bash
   cd fisio-flow-care
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Architecture
Fisio Flow Care is built using the following technologies:
- **Frontend:** React
- **Backend:** Node.js and Express
- **Database:** MongoDB

The architecture follows the Model-View-Controller (MVC) design pattern, providing a clear separation of concerns and enhancing scalability.

For more information, please refer to the [Wiki](https://github.com/Ewrogabriel/fisio-flow-care/wiki).
